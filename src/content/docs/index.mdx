## Installation

<img src="assets/title.jpg" alt="" style="width: 700px; display: block; margin: auto;"/>

RB Stat is a plugin allowing you to create RPG-typed characteristics (hunger, health, mana, experience, etc...) in few clicks, in C++ and BP, auto-replicating. Each stat can receive temporary bonus, which allow you to connect it easily to your buff/equipment system.

### Activate the plugin

<Steps>
	<ol>
		<li>
			<b>Get the plugin on Fab.</b>
			
			<LinkCard
				title="RB - Stat Plugin on Fab"
				href="https://www.fab.com/listings/8e6b48b6-74bc-4e91-be0a-ac63bc956c8e"
			/>
		</li>
			
		<li>
			<b>Add the plugin to your project.</b>
			
			In the Epic Launcher, go to Unreal Engine > Library.<br/>
			Scroll down to the Fab Library and add the plugin the your project.
		</li>

		<li>
			<b>Be sure it's enabled in plugin setting.</b>
			
			Go to Edit > Plugins.<br/>
			Check if RBStat is enabled.<br/>
			
			<img src="assets/enabled.png" alt=""/>
		</li>

		<li>
			<b>It's ready to use !</b>
		</li>
	</ol>

</Steps>
